## User Guides

### Purpose
*Step-by-step procedural instructions for end users to accomplish specific tasks*

### Structure
```markdown
# [Module] User Guide

## Introduction
[Who this guide is for]
[What users will accomplish]
[Prerequisites and assumptions]

## Getting Started
[Initial setup requirements]
[Access instructions]
[First-time user flow]

## Core Workflows

### [Primary Task Name]
[Brief description of when users would perform this task]

#### Prerequisites
[What users need before starting]
[Required permissions or access]

#### Step-by-step Instructions
1. [Action step with clear instruction]
   - [Sub-step if needed]
   - [Expected result or confirmation]

2. [Next action step]
   - [Screenshots or visual aids]
   - [Troubleshooting notes]

#### Expected Outcomes
[What users should see when complete]
[Next available actions]

## Advanced Operations
[Power user functionality]
[Administrative tasks]
[Customization options]

## Troubleshooting
[Common issues and solutions]
[Error message explanations]
[Support contact information]
```
